# منصة دورات القرآن - نظام إدارة الدورات القرآنية

## الغرض من المنصة

منصة **دورات القرآن** هي نظام إداري متكامل للدورات القرآنية، تهدف إلى تسهيل العمل الإداري ومتابعة الطلاب بشكل منظم واحترافي.

### الأهداف الرئيسية:

1. **رفع تقارير الإنجاز**
   - تسجيل الحفظ الجديد للطلاب
   - تسجيل المراجعة اليومية
   - توثيق الدرجات والتقييمات

2. **متابعة انضباط الطلاب**
   - تسجيل الحضور والغياب
   - تتبع المواظبة على الحلقات
   - رصد التأخير والانصراف المبكر

3. **تقييم مستويات الطلاب**
   - تحديد مستوى كل طالب (ممتاز، جيد جداً، جيد، مقبول، ضعيف)
   - تتبع الأخطاء المتكررة
   - عرض إحصائيات التقدم

## أنواع المستخدمين

### 1. الطالب
- عرض جدول الحلقات
- متابعة نسبة الحفظ والإنجاز
- الاطلاع على الدرجات والملاحظات

### 2. الشيخ / المعلم
- إدارة الحلقة وقائمة الطلاب
- تسجيل التسميع (حفظ جديد / مراجعة)
- رصد الدرجات وتسجيل الأخطاء
- رفع تقارير الجلسات

### 3. الإدارة
- توزيع الطلاب على المشايخ
- متابعة الحضور والغياب
- الاطلاع على تقارير الأداء الشاملة
- إدارة الحلقات والمسارات

### 4. ولي الأمر
- متابعة تقدم الأبناء
- الاطلاع على تقارير الحضور والدرجات

## الميزات الرئيسية

### تقارير الإنجاز
- تسجيل كل جلسة تسميع مع التفاصيل:
  - نوع التسميع (حفظ جديد / مراجعة)
  - المقدار (من سورة وآية إلى سورة وآية)
  - الدرجة من 100
  - ملاحظات الشيخ

### متابعة الحضور
- تسجيل حضور كل جلسة
- أسباب الغياب
- إحصائيات الانضباط لكل طالب

### تقييم الأخطاء
- تصنيف الأخطاء:
  - أخطاء التجويد
  - أخطاء التشكيل
  - النسيان
  - الإضافة
  - الإبدال
- تحديد شدة الخطأ (بسيط / جسيم / خطير)

### نظام التلعيب
- نقاط على الإنجازات
- أوسمة تحفيزية
- لوحة متصدرين

## قاعدة البيانات

### النماذج الرئيسية:

| النموذج | الوصف |
|---------|-------|
| CustomUser | المستخدم (طالب/شيخ/إدارة/ولي أمر) |
| StudentProfile | بيانات الطالب التفصيلية |
| SheikhProfile | بيانات الشيخ |
| Halaqa | الحلقة (الشيخ + مجموعة طلاب + الموعد) |
| Session | الجلسة (تاريخ + وقت + حالة) |
| Attendance | سجل الحضور |
| RecitationRecord | سجل التسميع (المقدار + الدرجة + الملاحظات) |
| RecitationError | تفاصيل الأخطاء |
| MemorizationProgress | تقدم الحفظ لكل سورة |
